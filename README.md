# SmartTransit
A C++ Qt-based transportation management system developed for NUST AI (2nd Semester).

## Features
- **Admin Portal:** Secure access to view and manage passenger data.
- **Passenger Portal:** User-friendly interface for booking trips.
- **Custom OOP Logic:** Utilizes Inheritance, Polymorphism .

## How to Build
1. Open `CMakeLists.txt` in Qt Creator.
2. Select the appropriate kit (MinGW/MSVC).
3. Build and Run.
